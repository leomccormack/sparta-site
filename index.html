Spatial Audio Real-Time Applications

A collection of open-source VST audio plug-ins for producing and reproducing spatial sound scenes.

Download Documentation Source code

Up to seventh-order

All Ambisonics related plug-ins support up to seventh order input/output.

Powered by SAF

All of the internal audio signal processing is developed using the open-source Spatial_Audio_Framework.

Personalised binaural listening

All plug-ins that reproduce sound over headphones allow you to load custom HRIR data via the SOFA standard.

Cross-platform

The plug-ins may be compiled for MacOS, Linux, or Windows, with CPUs based on either the x86 or ARM architecture.

Powered by JUCE

All graphical user interfaces are developed using the open-source JUCE framework.
