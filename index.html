Spatial Audio VST Plug-ins

A collection of open-source VST audio plug-ins for producing and reproducing spatial sound scenes.

Download Learn more Source code

Up to seventh order

All Ambisonics related plug-ins support up to seventh order input/output.

Powerered by JUCE

All graphical user interfaces are developed using the open-source JUCE framework.

Personalised binaural listening

All plug-ins that reproduce sound over headphones allow you to load custom HRIR data via the SOFA standard.

Cross-platform

The plug-ins may be compiled for MacOS, Linux, or Windows, based on either x86 or ARM.

Powerered by SAF

All of the internal audio signal processing is built on the open-source Spatial_Audio_Framework.

Dark mode

Switch to a low-light UI with the click of a button. Change colors with variables to match your branding.
